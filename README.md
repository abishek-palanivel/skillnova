## ğŸš€ Installation

### Prerequisites

- Python 3.8+ (3.13.3 recommended)
- Node.js 14+ (18.x recommended)
- PostgreSQL 12+
- Git

### Quick Start

```bash
# 1. Clone repository
git clone https://github.com/abishek-palanivel/skillnova.git
cd skillnova

# 2. Setup database
createdb skillnova
psql -d skillnova -f database/schema.sql

# 3. Start backend (Terminal 1)
cd backend
pip install -r requirements.txt
python app.py

# 4. Start frontend (Terminal 2 - new window)
cd frontend
npm install
npm start
```

### Access the Application

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
- **Admin Dashboard**: http://localhost:3000/admin
- **Mentor Portal**: http://localhost:3000/mentor

```env
# Email Configuration
ADMIN_EMAIL=your-email@gmail.com
ADMIN_APP_PASS=your-app-password

# Flask
FLASK_ENV=development
FLASK_DEBUG=True
```
